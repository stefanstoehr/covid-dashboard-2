Hinweise:

Die Begrifflichkeiten "Infekte" und "Infektionen" stehen für positive PCR-Tests. Die Begrifflichkeit "Genesen" bedeutet, dass eine Person nach einem positiven Befund nicht mehr PCR-positiv getestet wurde. Für Fragen zu den Daten des RKI finden Sie in deren FAQ-Bereich eine erste Anlaufstelle.

Die vom RKI zur Verfügung gestellten Fallzahlen für die Stadt Cottbus/Chóśebuz werden beim Seitenaufruf von einer REST API des RKI abgefragt. Die Zahlen auf Cottbus.de unterscheiden sich womöglich von den Zahlen des RKI (vgl. Robert Koch-Institut: COVID-19-Dashboard). Ursachen liegen evtl. in der abgefragten Datenquelle selbst als auch in zeitlichen Übermittlungsfehlern.

Die Balkendiagramme (hoverbar/antippbar) sind Auswertungen über die REST API und berücksichtigen Geschlecht und Altersgruppen. Die App repräsentiert Web-Technologien, hingegen muss die Richtigkeit der Statistiken von Data Scientists geprüft werden.

Der Quellcode dieser Anwendung steht auf Github öffentlich unter der Creative Commons Lizen CC-BY-SA zur Verfügung. Bei Hinweisen und Fragen melden Sie sich bitte.
